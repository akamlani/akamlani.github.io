    • Numenta Platform for Intelligent Computing (NuPIC)

    NuPIC is an open source community project focusing on a set of learning algorithms of how layers of neurons in the neocortex learn. Of particular focus is the key Hierarchical Temporal Memory (HTM) model. It's implementation is most closely modeled after layer 3 in the Neocortex. Reference the cortical learning algorithm (CLA) and NuPIC github repositories for more information.

    My interest followed based on the release of the the book "On Intelligence" by Jeff Hawkins initially released in 2005. Since then the research and product offering has evolved. The key differentiators are: Sparse Distributed Representations (SDRs), Temporal Inference, and Online Learning.

    This past November, I attended the Fall Hackathon 2013. This was the first real introduction to NuPIC for the general public in the form of a Hackathon. It consisted of a mix of informational sessions, presentations and presented hacks using the framework.

    As quoted from the Session:

    I think we had a nice mix of people at the event, e.g. from those focused on neuroscience biological aspects, to those just getting their feet wet, a few hackers applying NuPIC to some applications, and the great staff. Great new introductions, good conversations and informational sessions. Definitely an event worth attending!

    - Ari Kamlani

    Project #2: NuPIC WiFi Prediction RSSI Sampling

    http://github.com/akamlani/nupic_wifi
    Prediction of Wi-Fi RSSI samples to predict 'x' samples into the future. How does the RSSI value predict over time, presenting a repeated walking pattern. This demo was using Wi-Fi Direct and perform scans from a PC based on the advertised SSID and statistics, taking into account real world interference and physical barrier problems. It would be interesting to see how this implementation compares against a more accurate time pulse based solution not using RSSI (UWB, IEEE 802.15.4 with ranging methods).

    Electric Imp

    Electric Imp is emplowering the Internet of Things (IoT), connecting the physical world (Device) with the services world (Agent, Server). The user code is written in a minimal form of the Squirrel Language and debugged in their cloud based IDE. The device has an integrated ARM Cortex M3 Processor and WiFi in the form of SD Card w/Socket Module (Imp001) vs Soldered Down Model (Imp002). The WiFi configuration consists of a unique Proprietary soltuion for device setup, via a Blinkup Method. Reference some of their services and examples via their Github Repo.

    TI BeagleBone, BeagleBoard

    The TI BeagleBone Black edition is the newest model from TI. It no longer runs a TI OMAP Processor, like its predcecessor (White version). The "Black" version removed several onboard components in order to compete with the Raspberry Pi, e.g. USB JTAG, USB to Serial converter, etc. The development platform is based on the TI AM335x 1GHz ARM Cortex-A8 Processor and comes with Angstrom. However I much prefer Ubuntu console with a minimal Root FS (RFS).

    Gumstix

    Gumstix provide a Computer on Modules (COM) form factor. The model which I have worked on is the Overo Fire. It is based on a TI OMAP 3530 (ARM Cortex-A8) Application Processor, similar to the BeagleBoard (not the BeagleBone) earlier processors.
